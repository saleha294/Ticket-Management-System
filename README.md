# 🎟️ Console-Based Ticket Management System

### A C++ application for managing travel and event ticket bookings.

Built as a university project to demonstrate object-oriented programming, file handling, and efficient data management through a command-line interface.

</div>

---

## ✨ Features

- 🎫 Book tickets
- ✏️ Update booking information
- ❌ Cancel bookings
- 🔍 Search booking records
- 📋 View all bookings
- 💾 File-based data storage
- 🖥️ Menu-driven console interface
